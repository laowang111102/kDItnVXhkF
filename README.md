# 知识图谱问答系统 python1188

## 项目概述


## 技术栈

- 后端：Python
- 图数据库：Neo4j
- 机器学习：深度学习

## 功能模块

- 知识图谱构建
- 知识图谱可视化
- 问答系统开发

## 系统角色

- 知识图谱构建者
- 问答系统使用者

## 运行环境

- 操作系统：Linux或macOS（推荐），Windows支持但不保证最佳性能
- Python版本：3.6及以上
- Neo4j版本：3.5及以上

## 部署步骤

1. 安装Python环境
2. 安装并启动Neo4j数据库
3. 克隆项目代码到本地
4. 导入知识图谱数据到Neo4j
5. 启动问答系统服务
